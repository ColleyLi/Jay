专辑名称：霍元甲 EP

演唱歌手：周杰伦

发行时间：2006年1月20日

专辑语种：国语专辑

------------
专辑介绍：

周杰伦一直示国际巨星李连杰为他的偶像，李连杰与Jay惺惺相惜，为了李连杰主演的电影《霍元甲》(Fearless)，终于有机会合作，由周杰伦作主题曲及主唱。而Jay表示能为偶像主演的电影作曲及主唱主题曲，心情兴奋。与电影同名主题曲，希望透过一代中国武术宗师霍元甲的一生传奇及Jay的歌声，来宣扬积极向上的讯息。此曲及其MV均辑录于《霍元甲 EP》内。 

------------
专辑曲目: 

01. 霍元甲

02. 献世（Live）
